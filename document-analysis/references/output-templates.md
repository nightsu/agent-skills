# Document Analysis 输出模板

## 1. `general`

```markdown
# 摘要

## 核心摘要

## 关键要点

## 延伸思考（可选）
```

## 2. `requirements`

```markdown
# 摘要

## 文档类型

## 核心目标

## 关键需求点

## 缺失信息 / 风险点

## 优先级

## 待确认事项
```

## 3. `technical`

```markdown
# 摘要

## 文档类型

## 核心结论

## 关键设计 / 实现点

## 逻辑检查

## 风险点

## 优先级

## 待确认事项
```

## 4. `proposal`

```markdown
# 摘要

## 文档类型

## 方案概述

## 关键假设 / 前提

## 方案一致性检查

## 风险点

## 优先级

## 待确认事项
```
